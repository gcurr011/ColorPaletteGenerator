# ColorPaletteGenerator

## 🎨 Project Description
Color Palette Generator is a web application that generates a palette with 2-6 colors from user-inputted hex color values. 

This was built as a class project for COP 4814: Component-Based Software Development. 

The whole goal of this web-based application was to integrate the X-Color API to derive color palettes of the following types: 
- Complementary 🌸- Colors that are opposite each other on the color wheel (e.g. blue and orange). They create high contrast and make each other appear more vibrant when paired together.
- Analogous 🎡 - Colors that sit next to each other on the color wheel (e.g. red, red-orange, and orange). They create a harmonious, cohesive look and are often found in nature.
- Monochromatic 🪟 - Variations of a single color using different shades, tints, and tones (e.g. light blue, medium blue, dark blue). Creates a clean, unified aesthetic.
- Triadic 🔺 - Three colors evenly spaced around the color wheel (e.g. red, yellow, and blue). Offers strong visual contrast while still feeling balanced and vibrant.

## 🛠️ Tech Stack 
- **Python** - backend logic and API integration
- **HTML/CSS** - frontend and design interface
- **X-Color API** - color palette generation 
 
## 🚀 How It Works 
1. User enters a hex color code (example: '#E6E6FA')
2. The app sends the value to the X-Color API
3. User picks which type of palette they want returned (example: "Triadic")
4. A palette of the selected type is returned and displayed
